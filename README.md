# onion-cli
将工作中一些常用的命令整合成功一个 onion 命令

## 功能列表
### 开发
- [] user 获取测试环境用户信息
- [] rn RN 打包脚本

### 工具
- [x] ip          获取本机 IP 地址
- []  user         获取测试环境用户信息
- []  www          指定目录启动本地服务
- []  ssh          SSH 快速登录
- []  aqi          查询城市 AQI 指数
- []  url          URL 地址操作
- []  qr           生成和识别二维码
- []  rn           RN 功能支持
- []  color        色值转换
- []  yd           有道翻译
